`https://github.com/mahmoud-abouahmad/app`

`963956268336`

## ما تم تنفيذه

- إعادة تصميم شاشة الدخول والخلفيات والبطاقات والأزرار وحقول الإدخال.
- تحسين شكل جميع النوافذ المنبثقة ونوافذ الإضافة والتعديل والحذف.
- واجهة أستاذ متجاوبة:
  - قائمة جانبية على الهاتف.
  - شريط تنقل منظم على الشاشات الكبيرة وChrome.
- تحسين لوحة الطالبة وشريط التنقل السفلي.
- خط عربي Cairo.
- تحسين عرض اللوغو داخل التطبيق.
- أيقونة تطبيق مربعة مأخوذة من اللوغو المعتمد.
- شاشة بداية Android بألوان الهوية واللوغو.
- تحسين اسم وشكل نسخة الويب وشاشة التحميل في Chrome.
- تعديل وحذف الدروس.
- فتح روابط ملفات الدروس والفيديوهات من داخل التطبيق.
- تعديل وحذف النتائج.
- تعديل وحذف الحضور.
- تنبيهات داخل التطبيق مصنفة إلى:
  - تحفيز.
  - موعد مذاكرة.
  - تنبيه مهم.
  - واجب.
- تعديل وحذف التنبيهات.
- صفحة إعدادات متكاملة للأستاذ والطالبة.
- قسم تواصل يفتح واتساب على الرقم `0956268336` مع رسالة جاهزة.
- زر لنسخ رقم واتساب.
- رابط واتساب في شاشة الدخول وشاشة انتظار تفعيل الحساب.
- إنشاء Release APK شامل ونسخ أخف حسب معمارية الهاتف.
- فحص وتحليل واختبارات تلقائية قبل بناء APK.

## طريقة رفع النسخة

1. فك ضغط الملف المضغوط.
2. انسخ محتويات المجلد فوق ملفات المشروع الحالي مع الحفاظ على المسارات.
3. ارفع الملفات إلى فرع `main` في Commit واحد.
4. افتح تبويب **Actions** في GitHub.
5. افتح المسار **Build Final Release APK**.
6. انتظر نجاح المراحل الآتية:
   - Install packages
   - Generate application icon
   - Format source
   - Analyze source
   - Run tests
   - Build universal release APK
   - Build smaller release APKs
7. من قسم Artifacts حمّل:

`mahmoud-dyab-final-v1-apks`

## ملف APK المناسب للتجربة

ابدأ بالملف الشامل:

`mahmoud-dyab-app-release.apk`

وللهواتف الحديثة يمكن استخدام النسخة الأخف:

`mahmoud-dyab-arm64-release.apk`

إذا رفض الهاتف تثبيت النسخة فوق نسخة Debug القديمة بسبب اختلاف التوقيع، احذف النسخة القديمة ثم ثبّت نسخة Release.

## قائمة الاختبار بعد التثبيت

### حساب الأستاذ

- تسجيل الدخول.
- إضافة طالبة وتعديل بياناتها وتفعيلها أو إيقافها.
- إضافة درس ثم تعديله وحذفه.
- فتح رابط ملف أو فيديو مضاف إلى الدرس.
- إضافة نتيجة ثم تعديلها وحذفها.
- تسجيل حضور ثم تعديله وحذفه.
- إضافة الأنواع الأربعة من التنبيهات ثم تعديلها وحذفها.
- فتح صفحة الإعدادات وتجربة زر واتساب وزر نسخ الرقم.

### حساب الطالبة

- التأكد من ظهور دروس مجموعتها فقط.
- التأكد من ظهور نتائجها فقط.
- التأكد من ظهور حضورها فقط.
- التأكد من ظهور تنبيهات مجموعتها والتنبيهات العامة.
- تجربة رابط واتساب من الإعدادات.

## ملاحظة التوقيع

نسخة Release الحالية موقعة بمفتاح Debug لتسهيل التجربة المباشرة. قبل النشر الرسمي على Google Play يجب إنشاء مفتاح توقيع خاص وآمن للتطبيق.
